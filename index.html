FBX 資產管理器
選擇一個資料夾以瀏覽其中的 FBX 角色
會遞迴掃描子資料夾,自動配對相鄰的貼圖檔(png / jpg / webp / bmp / tga / dds)。 貼圖以「檔名」比對重導,因此 FBX 內記錄的原始絕對路徑失效也沒關係。
需要 Chrome / Edge(File System Access API)。其他瀏覽器會退回單次資料夾上傳模式。
